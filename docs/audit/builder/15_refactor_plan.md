# Plano de Refatoração


Plano de refatoração contínua para polir pequenos gaps identificados.

## 🛠️ 1. Melhoria de Roteamento de Links
- **PR-10.1: Dynamic Link Customizer**: Adicionar o campo `ctaUrl` na interface de blocos `BuilderBlock` e permitir que o administrador digite uma URL de destino personalizada para os botões do site, em vez de rolar de forma fixa para `#contact`.
- **PR-10.2: IA Image Enhancement**: Integrar sugestão de termos de busca de imagem baseada em inteligência artificial na aba de biblioteca Unsplash do MediaPicker.
